# 🐾 Pet Disease Detection System (Final Year Project)

An AI-based web application for detecting diseases in cats and dogs using Deep Learning and Computer Vision.

---

## 📌 Project Overview

This system allows users to:

- Upload a pet image
- Automatically detect whether the animal is a **Cat or Dog**
- Predict possible disease category
- View prediction confidence score
- Get basic health advice
- Find nearby veterinary clinics (Map integrated)

This project was developed as a Final Year Project (FYP).

---

## 🧠 AI Models Used

The system uses three trained deep learning models:

1️⃣ Animal Classification Model (Cat / Dog)  
2️⃣ Dog Disease Classification Model  
3️⃣ Cat Disease Classification Model  

Framework: **TensorFlow + Keras**

## 🚀 Installation & Setup Guide

### 🔹 Step 1 — Clone Repository

git clone https://github.com/Bilal-Sheikh-98/PET-Disease-Detection.git

---

### 🔹 Step 2 — Backend Setup

Go to backend folder:

cd backend

Create virtual environment:

python -m venv venv

Activate virtual environment:

**Windows:**

venv\Scripts\activate


Install dependencies:

pip install -r requirements.txt


Run backend:
venv\Scripts\activate
python app.py

Backend will run at:

http://127.0.0.1:5000


---

### 🔹 Step 3 — Frontend Setup

Open new terminal and go to frontend folder:

cd frontend


Run simple server:

python -m http.server 8000

Open browser:

http://localhost:8000/
